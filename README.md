## 🚀 О проекте

Habits — современное веб-приложение для отслеживания привычек,
анализа прогресса и ведения наблюдений с помощью таблицы записей и метрик.  

### Основные возможности
- 📌 Фиксация привычек с учётом типа и повторяемости  
- 📊 Метрики и аналитика  
- 📋 Таблица всех записей  
- 🧩 Управление группами и привычками  
- ⚙️ Глобальные настройки  
- 📈 Подготовка к визуализации данных (графики в будущем)

---

### Типы привычек
- ✅ **Обычные** — отмечаем выполнено/не выполнено  
- 📝 **Текстовые** — записи заметок или обьектов(названия книг и тому подобное)
- ⏰ **Временные** — фиксируем определённое время (например, время сна, тренировки)  
- 🔢 **Числовые** — фиксируем количество или значение (например, количество страниц, шагов, повторений)

### Повторяемость привычек
- Каждый день  
- Через день  
- Несколько раз в неделю  
- Несколько раз в месяц
